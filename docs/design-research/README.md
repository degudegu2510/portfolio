## AI × デザインプロセスの注目ポイント

各書籍で共通して AI 活用が期待できる領域：

- **ユーザーリサーチの自動化**：インタビュー文字起こし・テーマ抽出・ペルソナ生成
- **プロトタイピングの高速化**：Figma AI・v0 などによる UI 生成
- **仮説検証ループの加速**：Build-Measure-Learn を AI でショートカット
- **オブジェクトモデリング支援**：エンティティ抽出・関係性の整理
- **Opportunity Solution Tree の自動構築**：ユーザーデータから機会を自動分類
